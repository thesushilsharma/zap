# zap

This is a boilerplate project for building an Express.js application using TypeScript and ESLint.

## Tech stack

- Typescript
- Node.js
- NPM
- EJS (Embedded JavaScript) / TypeScript
- VS Code

## Prerequisites

- NPM Account

## Folder Structure
```TS
express-generator/
├── bin/
│   └── cli.ts
├── src/
│   └── lib/
│       ├── main.ts
│       ├── templates/
│       │   └── index.ts
│       └── utils/
│           ├── fileUtils.ts
│           ├── npmUtils.ts
├── package.json
├── tsconfig.json
└── README.md
```
